Features

Everything you need, nothing you don't

⚡

Auto-renders on open

Navigate to any .excalidraw URL and the viewer launches instantly.

🔍

Pan & zoom

Full Excalidraw view mode — explore large diagrams at any zoom level.

🔒

View-only

Diagrams are rendered read-only. Nothing is modified, uploaded, or tracked.

🌐

GitHub & GitLab

Supports blob pages and raw URLs on both GitHub and GitLab.

Installation

Up and running in 4 steps

1

Clone & install

Clone the repo and install dependencies with npm install.

2

Build

Run npm run build to generate the dist/ folder.

3

Load unpacked

Open chrome://extensions, enable Developer mode, click Load unpacked, select dist/.

4

Open any .excalidraw URL

Navigate to a .excalidraw file on GitHub or GitLab. Expeek opens the viewer automatically.

# Clone
git clone https://github.com/im-adarsh/expeek.git
cd expeek

# Install & build
npm install
npm run build

# Load dist/ as unpacked extension in chrome://extensions
